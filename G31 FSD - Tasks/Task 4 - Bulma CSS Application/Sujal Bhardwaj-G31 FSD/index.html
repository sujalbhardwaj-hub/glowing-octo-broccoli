What is Bulma CSS?

Bulma is a free and open-source CSS framework based on Flexbox. It provides ready-made classes that help developers build responsive and attractive websites quickly.


  • ✔ Easy to Learn
  • ✔ Responsive Design
  • ✔ Flexbox Based
  • ✔ No JavaScript Required
  • ✔ Modern Components

Installation

Add this CDN link inside your HTML file: 
<link rel="stylesheet"
href="https://cdn.jsdelivr.net/npm/bulma@1.0.2/css/bulma.min.css">
