Challenge Questions

28. What is the default value of the position property?

Answer: static

29. Which position type removes an element from the normal document flow?

Answer: absolute (and fixed).

30. Which position type keeps an element visible during scrolling?

Answer: fixed.

31. Which position type requires top, left, right, or bottom values to create a sticky effect?

Answer: sticky.

32. What happens when multiple elements have the same z-index value?

Answer: The element appearing later in the HTML is displayed on top.

33. When would you use absolute positioning instead of relative positioning?

Answer: When an element needs exact placement relative to its parent, such as badges and notifications.
